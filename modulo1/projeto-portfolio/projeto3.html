CSS

O CSS foi a minha mais recente descoberta no mundo da programação, com ele é possível fazer várias estilizações, colocar diferente fontes, arrumar nos mínimos detalhes de uma página, etc... CSS é uma ferramenta que ao mesmo tempo é simples e complexa, de fato, são muitos os comandos, as medidas e as formas que podemos editar uma página. O projeto CSS foi este próprio portfolio, que exigiu uma grande dedicação e colocou a prova meus conhecimentos.
