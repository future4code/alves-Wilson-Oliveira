JavaScript

O java script é uma ferramenta muito importante para fazermos a parte lógica de qualquer programa funcionar. Existem diversas funcionalidades que possibilitam uma infinidade de opções e meios para se chegar a um determinado resultado. Neste projeto, eu desenvolvi um jogo de Blackjack para html utilizando apenas o javascript, para isso foi necessário o conhecimento de alguns dos conceitos fundamentais do javascript, como variáveis, operadores lógicos e aritméticos, arrays, objetos e funções. Com esse projeto aprendi que o javascript é uma ferramenta muito complexa e que as possibilidades de criação com ele são infinitas.
